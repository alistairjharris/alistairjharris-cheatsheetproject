Positioning HTML using CSS

Declaration Role Variables Example
z-index The CSS z-index property specifies how far back or how far forward an element will appear on a web page when it overlaps other elements. Positive or negative integer values. Higher integers will render the element further forward on the website. element {
z-index: 10;
}
position Positioning in CSS provides designers and developers options for positioning HTML elements on a web page. Fixed
When the CSS position has a value of fixed, it is set/pinned to a specific spot on a page. The fixed element stays the same regardless of scrolling.		 element {
position: fixed
}
Absolute
The value absolute for the CSS property position enables an element to ignore sibling elements and instead be positioned relative to its closest parent element that is positioned with relative or absolute.		 element {
position: absolute
}
Relative
The value relative of the CSS position property enables an element to be positioned relative to where it would have originally been on a web page. 		element {
position: relative
}
Static
This is the default value of the position property. The element stays in the flow of the document and appears in the actual location where itâ€™s located in the markup. 		element {
position: static
}
Sticky
This value is a mix between the relative and fixed positions. The element is treated as relative until a specific point or threshold is met, at which point the element is then treated as fixed. 		element {
position: sticky
}
